## Grade Checker

A simple Rust exercise to practice Control Flow

## Concepts Learned
 - if
 - if else
 - else 
 - Comparison operators
 - Comparison operators
 - Conditional branching

## Grading rules

|   Score  |Grade| Status |
|---------:|:---:|:------:|
| 90 - 100 |  A  | Passed |
| 80 - 89  |  B  | Passed |
| 70 - 79  |  C  | Passed |
| 60 - 69  |  D  | Passed |
| 0  - 59  |  F  | Failed |
