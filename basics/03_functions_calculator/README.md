# Calculator
A simple Rust exercise to practice Functions.

## Goal

Build a basic Calculator using separate functions for each arithmetic operation.

## Concepts Learned
 - Function declaration
 - Function parameters
 - Return values
 - Explicit return with `return`
 - Implicit return
 - Function calls

## Operations

The calculator supports :
 - Addition
 - Subtraction
 - Multiplication
 - Division

## Output

========== Calculator ==========

20 + 10 = 30
20 - 10 = 10
20 * 10 = 200
20 / 10 = 2
